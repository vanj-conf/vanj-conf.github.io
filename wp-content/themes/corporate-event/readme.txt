== Description ==

Corporate Event is free WordPress theme for events, conferences, conclaves, and meetup. This theme is based on the Event and Conference Management WordPress Plugin, WPEventPartners. If you are the conferences organizers and looking to build one page conference website then this is the best theme for you. 




= Does this theme support any plugins? =

	Corporate Event includes support for WPEventPartners plugin.

== Installation ==
	
	1. In your admin panel, go to Appearance > Themes and click the Add New button.
	2. Click Upload and Choose File, then select the corporate-event.zip file. Click Install Now.
	3. Click Activate to use your new theme right away.
	4. Go to Customiser -> Home Page Setting -> Select - Static Front Page
	5. Install wpeventpartners wordpress plugin, then follow instruction from plugin.
	6. Create categories, pages and post.
